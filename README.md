## 📦 Installation

To replicate and run this project, follow these steps using Windows PowerShell or Command Prompt:

```bash
# 1. Install Node.js (if not installed)
winget install OpenJS.NodeJS.LTS

# 2. Clone the repository
git clone https://github.com/yourusername/firstattempt2026_Grande.git

# 3. Navigate into the project folder
cd firstattempt2026_Grande/alumni-connect

# 4. Install dependencies
npm install

# 5. Run the development server
npm run dev
```
